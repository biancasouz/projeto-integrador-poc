# Projeto Integrador - Vital Saúde

## Sobre o projeto
Sistema web desenvolvido como Prova de Conceito (PoC) para gerenciamento de exames ocupacionais.

## Funcionalidades
- Cadastro de pacientes
- Cadastro de empresas
- Agendamento de exames
- Emissão simulada de ASO

## Tecnologias utilizadas
- Python
- Flask
- HTML
- CSS
- GitHub

## Objetivo
Demonstrar a viabilidade de uma solução tecnológica para clínicas de saúde ocupacional.
